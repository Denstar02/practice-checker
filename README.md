# Анализатор отчётов по педагогической практике

Веб-приложение для автоматической проверки отчётов по педагогической практике с помощью ИИ (IO Intelligence API).

## Структура проекта

```
practice-checker/
├── server.py          # Flask-сервер (бэкенд)
├── requirements.txt   # Python-зависимости
├── README.md          # Этот файл
└── static/
    └── index.html     # Веб-интерфейс (фронтенд)
```

## Быстрый старт

### 1. Установите Python 3.8+

Скачайте с https://python.org если ещё не установлен.

### 2. Установите зависимости

```bash
cd practice-checker
pip install -r requirements.txt
```

### 3. Запустите сервер

```bash
python server.py
```

### 4. Откройте в браузере

```
http://localhost:5000
```

## Настройка API-ключа

API-ключ IO Intelligence уже встроен в `server.py`. Если хотите использовать свой:

**Вариант 1** — переменная окружения:
```bash
export IO_API_KEY="ваш-ключ"
python server.py
```

**Вариант 2** — отредактируйте `server.py`, строка `IO_API_KEY = ...`

## Как это работает

1. **Фронтенд** (`static/index.html`) — чистый HTML/CSS/JS, без фреймворков
   - Загрузка файлов .docx, .pdf, .txt
   - Извлечение текста прямо в браузере (mammoth.js для .docx)
   - Отображение результатов анализа с Bento Grid

2. **Бэкенд** (`server.py`) — Python/Flask
   - Принимает текст от фронтенда
   - Проксирует запрос к IO Intelligence API (ключ остаётся на сервере)
   - Возвращает структурированный JSON с оценками

3. **ИИ-анализ** — IO Intelligence API (Llama, DeepSeek, Qwen, Mistral)
   - Проверяет отчёт по 4 критериям
   - Выставляет баллы 0–100 по каждому критерию
   - Формирует рекомендации по доработке

## Критерии проверки

1. **Нормативная документация** — анализ ФГОС СПО, ОПОП, учебного плана, рабочих программ
2. **Место дисциплины** — вклад в компетенции, курс, объём нагрузки, таблица
3. **Междисциплинарные связи** — предшествующие/сопутствующие/последующие связи, схема
4. **Внутридисциплинарные связи** — анализ КТП, связи темы, схема

## Деплой на сервер

Для размещения в интернете можно использовать:
- **Railway** / **Render** — бесплатный хостинг Python-приложений
- **VPS** (любой) — `pip install gunicorn && gunicorn server:app`
- **Docker** — добавьте Dockerfile

## Технологии

- Python 3 / Flask
- Vanilla HTML / CSS / JS
- IO Intelligence API (open-source LLM)
- Mammoth.js (чтение .docx в браузере)
