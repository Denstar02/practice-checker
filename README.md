# Анализатор отчётов по педагогической практике

Веб-приложение для автоматической проверки студенческих отчётов по педагогической практике.

## Быстрый старт

Убедитесь, что у вас установлен Python (версии 3.8 или выше).

1. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```

2. Запустите сервер:
   ```bash
   python server.py
   ```

3. Откройте приложение в браузере: `http://localhost:5000`

## Настройка

Проект использует API от IO Intelligence, ключ уже встроен в код. 
Если вы хотите использовать собственный API-ключ, задайте его перед запуском через переменную окружения `IO_API_KEY` или укажите напрямую в файле `server.py`.

## Что проверяется

Вы можете загружать отчёты в форматах `.docx`, `.pdf` и `.txt`. Приложение оценивает их по следующим критериям:
- Использование нормативной документации (ФГОС, рабочие программы).
- Определение места дисциплины в учебном процессе.
- Описание междисциплинарных связей.
- Описание внутридисциплинарных связей и КТП.

По каждому из пунктов выставляются баллы (от 0 до 100) и даются рекомендации по доработке.

## Размещение в сети

Систему легко развернуть на любом хостинге, поддерживающем Python (например, Railway, Render или обычный VPS с Gunicorn), либо упаковать в Docker.
